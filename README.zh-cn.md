# 草料扫码器 Skills

草料扫码器应用的 Agent Skills 集合。通过 Agent 交互完成扫码登录、内容添加等操作。

---

> **⚠ 使用前请阅读**
>
> 1. **Token 安全**：登录授权后，Token 会以文件形式保存在本机。Token 拥有账号的完整操作权限，请勿将该文件分享给他人或提交到代码仓库，同时明确其他 Skills / Rules 的访问权限，避免 Token泄漏。
> 2. **谨慎操作**：Skills 的执行效果与所使用的模型能力强相关。**删除记录等操作不可逆**，执行前请仔细确认，避免误操作导致数据丢失。

---

## 安装

### 方式一：通过 npx skills 安装（推荐）

安装所有 Skills：

```bash
npx skills add caoliao/deepscan-skills --all
```

### 方式二：克隆仓库到本地

```bash
git clone https://github.com/caoliao/deepscan-skills.git
```

### 依赖安装

- Python 3.8+
- 安装依赖：`pip install -r requirements.txt`

## 支持的功能

| 功能 | 触发示例 |
|------|---------|
| 扫码登录 | 「登录草料扫码器」、「重新登录」 |
| 新建扫码本 | 「新建扫码本」、「创建一个叫 XX 的扫码本」 |
| 查看 / 切换扫码本 | 「我的扫码本」、「切换到 XX」、「显示扫码本列表」 |
| 开启新批次 | 「开启新批次」、「重新开始」、「新建批次」 |
| 添加扫码记录 | 直接发送文本或上传含二维码的图片 |
| 查看记录列表 | 「查看记录」、「历史扫码」、「扫了什么」 |
| 删除记录 | 「删除最新记录」、「删除第 2 条」 |
| 导出记录 | 「导出扫码本」、「导出 Excel」、「下载数据」 |


## 快速上手

以下是一段典型的对话流程，按步骤操作即可：

---

**第一步：登录草料扫码器**

> 「登录草料扫码器」

Agent 会生成一个登录二维码，用微信扫码后自动完成授权。

---

**第二步：选择要添加记录的扫码本**

> 「显示我的扫码本」/ 「切换到 XXX 扫码本」

Agent 会列出你的所有扫码本，告诉它选哪一个即可。

---

**第三步：添加扫码记录**

直接发送文本或含二维码的图片：

> 「https://cli.im」
>
> （或上传一张带有二维码的图片）

文本直接作为扫码内容添加；图片会自动解码二维码后添加记录。

---

**第四步（可选）：开启新批次**

如果需要区分不同批次的记录：

> 「开启新批次」

新批次开启后，后续添加的记录会归入新批次，便于分组管理。

---

更多管理功能（记录导出、删除、历史查看等）请前往「**草料扫码器**」微信小程序操作。
